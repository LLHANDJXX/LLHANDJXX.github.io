oracle账号锁定解决

Windows环境下

1.以管理员身份登录数据库：

  • 打开命令行工具或终端。

  • 设置ORACLE_SID环境变量。

    1. 临时设置（仅对当前命令窗口生效）

  • 尝试以SYSDBA权限登录：sqlplus / as sysdba

2.解锁SYSTEM账号：

  • 登录成功后，执行以下SQL命令来解锁SYSTEM账号：ALTER USER system ACCOUNT UNLOCK;
  • 如果您同时也需要重置SYSTEM账号的密码，可以使用如下命令：ALTER USER system IDENTIFIED BY 新密码 ACCOUNT UNLOCK;，请将新密码替换为您想设置的密码。

3.验证和后续步骤：

  • 确认账户状态：

  • 解锁后，尝试登录SYSTEM账号，以确认账号已成功解锁。
